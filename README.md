### Step-by-Step Guide

#### Step 1: Prepare PC Files
1. Download both packages above.
2. Extract the **Main Bypass Package** directly to `C:\` root.
3. Folder must be named exactly **`PHP`** → `C:\PHP`
4. Extract the **Cracked Mi Unlock Tool** anywhere (recommended: Desktop).

#### Step 2: Install Modified Settings App
1. Go to `C:\PHP\Set Apps\`
2. Copy the correct APK to your phone:
   - **Android 13** → Use Android 13 APK
   - **Android 14** → Use Android 14 APK
3. On phone: Open File Manager → Install the APK.

#### Step 3: Enable USB Debugging
1. Open the newly installed **Settings** app.
2. Go to **Additional Settings > Developer Options**.
3. Enable **USB Debugging** (accept the 10-second warning).
4. Connect phone to PC and authorize USB debugging.

#### Step 4: Run Bypass Script
1. Open `C:\PHP\` folder.
2. Double-click **`bypass.cmd`**.
3. Command window opens → Click **OK** on the pop-up when prompted.

#### Step 5: Bind Account
1. On phone: Developer Options → **Mi Unlock Status**.
2. Turn **OFF Wi-Fi**, turn **ON Mobile Data**.
3. Tap **Add account and device**.
4. PC may show **"Account Bound Successfully"** or show an error. **In both cases the account is successfully bound.**

#### Step 6: Restore Original Settings App
1. Long-press Settings icon on home screen → **App Info**.
2. Tap **Uninstall Updates** to restore stock Settings app.

#### Step 7: Instant Unlock with Cracked Tool
1. Power off your phone.
2. Hold **Volume Down + Power** to enter **Fastboot Mode**.
3. Go to the extracted **Cracked Mi Unlock Tool** folder.
4. Double-click **`batch_unlock.exe`** to run the tool.
5. Log in with your Mi Account:
   - If normal login fails, use **QR Code Login** option.
6. Connect phone via USB.
7. Click **Unlock**.
8. Tool will bypass the timer instantly and unlock the bootloader (5-10 minutes total).
9. Device will factory reset automatically.

**Bootloader is now unlocked.**

### Alternative Unlock (If Cracked Tool Doesn't Work)
Use the **official Mi Unlock Tool** with the same steps above. It will still unlock successfully.

### Troubleshooting
- Binding fails: Restart `bypass.cmd` and retry.
- Tool not detecting device: Install drivers from `C:\PHP\Drivers\`.
- Stuck at 50%: Reboot to fastboot and retry.
- Wrong APK: Use the one matching your Android version.
